Segnalazioni & Feedback

Z-Transit usa GitHub Issues per segnalazioni e suggerimenti.
Il sistema è pubblico e trasparente. È richiesto un account GitHub gratuito.

🐛 Bug

Qualcosa non funziona? Apri una issue con:

  • Cosa è successo e cosa ti aspettavi
  • Dispositivo e browser usati
  • Quando si verifica (sempre, sporadicamente, data/ora)

Segnala un bug

💡 Suggerimenti

Hai un'idea? Descrivila brevemente: cosa vorresti, perché sarebbe utile, come la immagini (opzionale).

Suggerisci un miglioramento

🕐 Orario non corretto

Prima di segnalare, verifica l'orario ufficiale su Movibus o Trenord. Se c'è discrepanza, indica linea, fermata, orario mostrato e orario corretto.

Segnala orario errato

Note

  • Le issues sono pubbliche — non inserire dati personali sensibili.
  • Il progetto è gestito da un singolo sviluppatore nel tempo libero. I bug critici hanno la priorità.
  • Contributi al codice benvenuti via fork + Pull Request.
  • Scrivi pure in italiano.
